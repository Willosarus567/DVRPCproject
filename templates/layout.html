Read Me:

To use this website please read the following steps:

  1. To access key mined data, press DVRPC Bike Data.
  2. To get back to the main page of S.I.P. click Home option on the navbar, Standard Information Page option on the navbar will result in a default page.
  3. To view DVRPC's main website click the first option on the main navbar's rightside menu.
  4. To access information about the weather or public transportation in the philadelphia area, click either of the two options next to DVRPC Main Website.
  5. This website is responsive on all screen sizes but to get the best viewing experience, please use mobile or desktop view.
